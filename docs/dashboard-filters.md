# Dashboard Filters

TASK-008B добавляет в dashboard sample-концепт гибких поисковых профилей и фильтров. Это не scraping и не реальный URL parsing: фильтры работают только по текущим sample-объектам.

## Профили

Dashboard поддерживает профили:

- `Коммерция 100–400 млн`;
- `Малые помещения до 30 млн`;
- `Офисы 30–150 млн`;
- `Помещения с арендаторами`;
- `Пользовательский профиль`.

Выбор профиля обновляет стартовые значения фильтров. Фильтры больше не жестко привязаны к диапазону 100–400 млн.

## Фильтры

Панель фильтров включает:

- цена min/max;
- цена за м² min/max;
- площадь min/max;
- этаж min/max;
- только первый этаж;
- локация / район / метро;
- источник;
- категория объекта;
- тип сделки;
- рекомендация;
- минимальный инвестиционный рейтинг;
- максимальный риск;
- минимальное качество данных;
- наличие арендатора;
- только федеральный арендатор;
- минимальная доходность;
- минимальная электрическая мощность;
- минимальный год здания;
- только с фото;
- только с недостающей информацией;
- уровень поддержки рынка.

## Результаты

Dashboard показывает:

- сколько объектов видно;
- сколько всего объектов входит в выбранный профиль;
- сколько скрыто текущими фильтрами;
- счетчики `Покупать`, `Изучить подробнее`, `Не рекомендую`.

## Воронка intake

Для активного профиля показывается intake funnel:

- последний запуск;
- следующий запуск;
- интервал обновления;
- лимит объявлений на источник;
- разбивка по источникам;
- разбивка по локациям.
